# Document & Story Templates

Standard blueprints to save structure tokens.

## Template 01: Functional User Story
```markdown
# STORY-[ID]: [Brief Title]
**As a** [User Role]
**I want to** [Actionable Objective]
**So that** [Business Benefit]

### Acceptance Criteria
- [] Scenario 1: [Given/When/Then]
```

## Template 02: Architectural Decision Record (ADR)
```markdown
# ADR-[ID]: [Decisive Title]
- **Status**: [Proposed | Accepted | Superseded]
- **Context**: [Problem Definition]
- **Decision**: [Adopted Solution & Trade-offs]
- **Consequences**: [Long term impact]
```
